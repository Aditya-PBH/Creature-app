# Creature-app
Creature is a modern social media application inspired by Instagram. It includes features like Stories (24hr), Reels with auto-play, real-time chat using Socket.io, user authentication, follow system, and push notifications. Built using React Native, Node.js, Express, and MongoDB.
# 🧬 Creature App

A full-stack social media application inspired by Instagram.

---

## 🚀 Features

- 📸 Stories (24hr auto delete)
- 🎬 Reels (Auto-play video feed)
- 💬 Real-time Chat (Socket.io)
- ❤️ Like & Comment system
- 👥 Follow / Unfollow users
- 🔔 Push Notifications (Firebase)
- 🔐 Authentication (JWT)

---

## 🛠️ Tech Stack

**Frontend:**
- React Native
- Expo

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Atlas)
